== Installation ==

= Installation =
1. In your WordPress Dashboard go to "Plugins" -> "Add Plugin".
2. Search for "My-CoolPay - Payment gateway for WooCommerce ".
3. Install the plugin by pressing the "Install" button.
4. Activate the plugin by pressing the "Activate" button.
5. Open the settings page for WooCommerce and click the "payments" tab.
6. Click on the sub tab for "My-CoolPay - Payment gateway for WooCommerce".
7. Configure your My-CoolPay Gateway settings.


= Recommended Requirements =
* WordPress version 5.5.1
* PHP version 7.2.24 or greater.
